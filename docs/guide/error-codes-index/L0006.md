# L0006

## Description

Invalid character for float / integer.

## Explanation

The character is not a valid character for a float / integer. Usually this is caused by a missing delimeter.

## Example

```
a = 1.a
```

## Example Error

```
--> main.exl
1 | a = 1.a
          ^ error[L0006]: invalid character for float
```
