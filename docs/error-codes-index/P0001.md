## Description

Expected 1 parser token before assignment, got {x} [^1].

## Explanation

The parser expects one parser token before an assignment.

## Example

```
a b = 1
```

## Example Error

```
--> ../../programs/test.exl
1 | a b = 1
        ^ error[P0001]: expected 1 parser token before assignment, got 2
```

[^1]: x = 0 || x > 1
