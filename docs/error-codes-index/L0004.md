## Description

Invalid escape sequence.

## Explanation

The lexer found an invalid escape sequence in a character / string literal.

## Example

```
a = '\a'
```

## Example Error

```
--> ../../programs/test.exl
1 | a = '\a'
         ^^ error[L0004]: invalid escape sequence
```
