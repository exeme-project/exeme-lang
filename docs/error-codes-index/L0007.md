## Description

Too many decimal points for float.

## Explanation

The number of decimal points for a float is limited to 1.

## Example

```
a = 1.1.1
```

## Example Error

```
--> main.exl
1 | a = 1.1.1
           ^ error[L0007]: too many decimal points for float
```
